PIC (Phylip) tab
This tab enables users to conduct independent contrast analysis using the Contrast tool from the PHYLIP package (http://evolution.genetics.washington.edu/phylip/doc/contrast.html).

To begin, users must choose <Add Job> from the <File> menu.
This brings up a “Configure New Job” window with the following options:

• Select Tree(s)-Select to find and choose trees from available files containing tree data for use in analysis. Selection of multiple trees is permitted for analysis as long as corresponding traits exist.

Selection of multiple tree files is enabled by the following actions:

    For Mac users: select initial file and hold <command> key while selecting subsequent files

    For PC users: select initial file and hold <control> key while selecting subsequent files.


• Select Traits(s)-Select to find and choose traits from available files containing trait data for use in analysis. All trait data that is to be used must be contained in one trait file. Selection of more than one trait file is not supported.

    *Notes:
  • if a file contains both tree and trait data, it will be available in both listings for trees and traits
    • if a file contains multiple trees, all trees will be selected for analysis.

• Select Params- Select to select which what details are desired in the output is desired. Available parameters are to display contrasts and display correlations and regressions.  (Future functionality will allow for the display of data sets selected for input). Selection of these parameters will enable the selected items for display on output. Deselection of these parameters will hide these details from the user on output. It is not possible to edit a job once it has been run to change these parameters.

The above items are enabled by default. There is no order required for selection of items within these panels.

Upon selection of tree and trait data, the following panels are enabled

  • Reconcile-Select to verify species and traits are correctly matched. Each species on the tree must have corresponding trait data for the application to work.
    • Trait data in excess of what is listed on the tree will be ignored in the analysis.
    • Tree data in excess of the trait listing must be edited from the tree in the editor tab. This requires exiting the wizard and opening the file in the editor, saving changes to the tree (save will replace previous file, save as will allow for saving as a new file) and then selection of the edited tree for analysis with a new job creation.

Once the tree and trait data have been selected, the user has several options:
        To utilize the reconcile functionality for tree data, select two species                             from panel with tree data and select "swap tree species"

     To utilize the reconcile functionality for trait data, select two species                                 from panel with trait data and select "swap trait species"

        Notes:
    • Swapping is able to be performed for one set of species at a time. Multiple changes require multiple swaps.
    • Exact matches of species names is not required. In the event of misspellings or different labels, simply matching tree species with trait intended to be used for analysis is all that is necessary.
    
Selection of apply in center of the bottom                     task bar is the indication that all species are reconciled and the data is prepared for                  the analysis.
 
  • Confirm-this panel becomes enabled only when data is reconciled and there is a one for one match of trait species for tree species. In the event of multiple tree selection, species will only be displayed once in the tree table on reconciliation tab, thus traits will be matched to all trees.
    • Confirm will not be enabled if there are tree species in excess of trait data. Trees will need to be edited in the editor tab, saved as a new file and then re selected in the wizard for analysis.
    • In the event that there are traits in excess of tree species, the traits in excess will be ignored in the analysis. 
    • Confirm will redisplay the tree file name selected, the trait file name selected and the parameters selected.
    • Selection of Save in this panel enables a window where the job can be named. 
      •  Names are limited to 250 characters that are alphanumeric. Only allowed non-alphanumeric characters are dash (-), underscore (_) and period (.)


Job Grid
Upon confirmation of job and naming, job will be entered into the Job grid on the contrast tab and saved for run at a time indicated by the user. The job grid contains a listing of all jobs run by the user or jobs in preparation for run. Jobs will remain in this grid until removed by the user.


  • Delete- allows for deletion of job from grid. 
  • Start- allows for submission of job for analysis. 
  • Refresh- refreshes grid for more immediate update of status.
  • Download Result- allows for download of result to a user's local environment.


Outputs of the contrast analysis will be placed in the original folder titled Data for visualization, even if the folder is renamed.
Outputs can be visualized by right clicking the file and selecting edit.
